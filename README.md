
Репозиторий для выпуска тестовых версий, анонсов будущих обновлений и экспериментов   
Здесь я тестирую новые функции, интерфейсы и ссылки, прежде чем перенести их в стабильный релиз.

## 🎯 Что здесь происходит

- Публикуются **тестовые версии** программы, которые могут быть нестабильными.
- Размещаются **анонсы** грядущих изменений
- Эксперименты с интерфейсом, новыми дистрибутивами, проверкой хешей.
- Всё, что пока не готово для основного репозитория.

## ⚠️ Предупреждение

- Код из этого репозитория может содержать ошибки, вылеты и некорректно работать.
- **Не используйте тестовые версии для реальной загрузки ISO-образов** – для этого есть [основной проект](https://github.com/winintern/ubuntu-iso-downlouder).
- Если вы нашли баг или есть идея – создавайте Issue.
